# plugin_test

插件编写学习

## 安卓端代码编写

在安卓目录下，点击其中一个文件选择在Android studio中编写即可

## iOS端代码编写

无法直接在iOS目录下选择Xcode打开，需要打开example项目中的iOS部分，然后找到Pods->Development Pods 找到对应的文件进行编写
